# IRB and Session Packager

## Overview
This packager creates standard operating procedures, IRB appendices, and BIDS event templates for imaging sessions.

## Setup
Install dependencies using:

```bash
pip install -r requirements.txt
```

Run the development server with:

```bash
uvicorn app.main:app --reload
```

## Test Data
A sample dataset is provided in the `data` folder to help you get started with input formats. You can replace it with your own data.
